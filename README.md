## 🎯 Problem Statement

With the rise of AI tools like ChatGPT and Copilot, distinguishing between **human-written** and **AI-generated** code has become increasingly difficult.

> ❓ *Is this code written by a developer or generated by AI?*

This system solves that problem using **automated repository analysis + AI classification**.

---

## 🚀 What This Project Does

✔ Analyze any GitHub repository
✔ Extract and filter relevant source files
✔ Perform **bulk AI analysis using background jobs**
✔ Classify each file as:

* 🤖 AI Generated
* 👨‍💻 Human Written

✔ Provide:

* Confidence Score
* File-level insights
* Repository-level summary

## 🏗️ System Architecture
```text
User Input (GitHub URL)
        ↓
Analysis Orchestrator
        ↓
Redis Queue System
        ↓
Repository Cloning
        ↓
File Filtering Engine
        ↓
Batch Processing Jobs
        ↓
AI Detection Layer (LLMs)
        ↓
Database Storage
        ↓
Frontend Dashboard
```

## 📦 Installation

```bash
git clone https://github.com/your-username/ai-code-analyzer.git
cd ai-code-analyzer

composer install
cp .env.example .env
php artisan key:generate
```

---

## ⚙️ Environment Setup

```env
QUEUE_CONNECTION=redis
REDIS_HOST=127.0.0.1

GITHUB_TOKEN=your_token

GROQ_API_KEY=your_key
DEEPSEEK_API_KEY=your_key
GEMINI_API_KEY=your_key
XAI_API_KEY=your_key
```

---

## ▶️ Run Application

```bash
php artisan serve
php artisan queue:work
```
